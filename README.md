# LabelMinus（工程中）

这是一个通过面向Gemini编程，基于 C# WinForms 开发的轻量级图像标注工具。在实现老前辈LabelPlus的基本功能的基础上进行了一定的优化。

## ✨ 核心功能特性

### 1. 标记模式
左键创建标注
右键标注可直接删除
### 2. 文校模式
鼠标悬停在标注上可直接选中标注并开始编辑，同时显示文本内容
### 3. 识别模式
给图片截图自动跳转字体识别网站，新窗口左侧显示截图，可直接拖入网站进行识别

### 4. 图校模式

* **压缩包直接打开**：支持.zip,.7z,.rar格式的压缩包直接打开图片，省去了解压操作与存储空间焦虑
* **图像同步显示**：两侧图片同步显示同一区域，帮助你找不同

## 🚀 快速开始

1. **新建翻译**：选择包含图像的文件夹，创建翻译文档。
2. **打开翻译**：选择翻译文档，自动加载所在文件夹的图片极其标注
3. **图片标注**：在图片上点击左键可打点
4. **保存翻译**：点击对应的分组按钮，标注将自动切换到对应颜色。
5. **导出翻译**：根据需要选择保存翻译/另存为翻译/导出修改文档等操作
