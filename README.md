# LabelMinus（页面建设中）

这是一个通过面向Gemini编程，基于 C# WinForms 开发的轻量级（>150M）图像标注工具。在实现老前辈LabelPlus的基本功能的基础上进行了一定的优化。

兼容原本LabelPlus的翻译文本导入导出。

## 🚀 快速开始

1. **新建翻译**：选择包含图像的文件夹，创建翻译文档 or **打开翻译**：选择LabelPlus翻译文档，自动加载所在文件夹的图片极其标注
2. **图片标注**：在图片上点击左键可打点，右键可删除已有标注
3. **保存翻译**：保存当前的翻译进度
4. **导出翻译**：根据需要选择保存翻译/另存为翻译/导出修改文档等操作

## ✨ 核心功能特性

### 1. 标记模式
* **图片显示**：鼠标滚轮缩放、拖动图片
* **创建标注**：左键点击创建标注，右键点击删除标注
* **标注显示**：图片界面实时显示标注文本内容


### 2. 文校模式
* **简洁显示**：图片界面现在仅显示标注序号，小窗显示翻译内容
* **标注拖动**：鼠标现在可以拖动标注位置
* **快捷编辑**：鼠标现在悬停在标注序号上会直接选中标注

### 3. 识别模式
给图片截图自动跳转字体识别网站，新窗口左侧显示截图，直接拖动图片进入网站识别

### 4. 图校模式

* **压缩包直接打开**：支持.zip,.7z,.rar格式的压缩包直接打开图片，也支持直接自选图片与图片文件夹，省去了解压操作与存储空间焦虑
* **图像同步显示**：两侧图片同步显示同一区域，帮助您找不同
* **截图至剪贴板**：截下对比图并直接复制到剪贴板中，方便您向嵌字者反馈
* **截图编辑**：鼠标悬停在底部缩略图显示截图大图，可直接红笔标记，右键清空标记
